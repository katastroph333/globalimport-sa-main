# GlobalImport S.A. Project

## Overview
GlobalImport S.A. is a fictitious company that provides a platform for international trade and import services. This project includes a responsive website that showcases the company's services and allows users to register for more information.

## Project Structure
```
globalimport-sa
├── css
│   └── styles.css        # Styles for the website
├── js
│   └── script.js         # JavaScript functionality
├── index.html            # Main webpage with navigation
├── registro.html         # Registration form for users
└── README.md             # Project documentation
```

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the main page.
3. Use the navigation bar to access different sections of the website, including the registration form.

## Features
- Responsive navigation bar with a hamburger menu for mobile devices.
- Links to key sections: Inicio, Servicios, Contacto, and Registro.
- User-friendly registration form in `registro.html`.
